# 📊 Teoría y Práctica de Pandas

Este repositorio contiene mi progreso y ejercicios prácticos en el dominio de la librería **Pandas** para el análisis de datos con Python. El objetivo es consolidar conocimientos en manipulación, limpieza y exploración de grandes volúmenes de datos.

## 🚀 Contenido del Repositorio

El estudio está estructurado siguiendo los pilares fundamentales del análisis de datos:

1.  **Introducción a Pandas:** Conceptos básicos y configuración del entorno.
2.  **Estructuras de datos:** Dominio de *Series* y *DataFrames*.
3.  **IO - Lectura y Escritura:** Trabajo con archivos CSV, Excel, SQL y JSON.
4.  **Exploración:** Primer contacto con los datos (`head`, `info`, `describe`).
5.  **Selección y Filtrado:** Uso de `loc`, `iloc` y filtros booleanos.
6.  **Limpieza de Datos:** Manejo de datos faltantes (*NaN*) y duplicados.
7.  **Transformaciones:** Aplicación de funciones y mapeo de datos.
8.  **Ordenamiento:** Reordenamiento de índices y valores.
9.  **Agregación:** Agrupamiento de datos con `groupby` y funciones de agregación.
10. **Uniones:** Combinación de DataFrames con `merge`, `join` y `concat`.
11. **Series Temporales:** Manejo de fechas y periodos de tiempo.
12. **Visualización:** Gráficos rápidos integrados con Matplotlib.
13. **Optimización:** Buenas prácticas y rendimiento en grandes datasets.
14. **Proyectos Finales:** Resolución de casos prácticos reales.

## 🛠️ Tecnologías Utilizadas

*   **Python 3.x**
*   **Pandas**
*   **NumPy** (como base para las estructuras de datos)
*   **Git/GitHub** para el control de versiones
